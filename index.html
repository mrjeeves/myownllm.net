Three steps. No accounts, no terminal commands required.

    Download & open

    One click on the download button below. Run the installer. Open MyOwnLLM like any other app on your computer.

  2. 2

    It picks the right brain for your machine

  3. 3

    Type a message. Drop in a photo. Hit record on a meeting. From here on, MyOwnLLM updates itself in the background — you never have to come back to "manage" it.

Things people ask

Is this really free?

Yes. MyOwnLLM is open source under the MIT licence — that means you can use it forever, even commercially, even offline, even after this project is no longer maintained. There's no premium tier, no account, no telemetry by default.

Does it work without internet?

Yes — after the first download. The first time you run it, MyOwnLLM downloads the AI model itself (typically 3–15 GB). After that it runs entirely offline. You only need internet again if you want to update the app or switch to a different model.

Do I need a fancy graphics card?

No. MyOwnLLM works on a stock Mac, a regular Windows laptop, even a Raspberry Pi 4 with 2 GB of RAM. It picks a model your machine can actually run. If you do have a GPU it'll happily use it to go faster.

Will it slow my computer down?

Only when you're actively chatting — and even then it's bounded by what your computer can comfortably handle, because the model is chosen to fit. Idle, MyOwnLLM uses no meaningful power.

How much disk space does it need?

The app itself is small (under 50 MB). The model it downloads on first run is the big bit — typically 3 GB on a Pi or small laptop, up to about 15 GB on a powerful machine. Models you stop using are cleaned up automatically.

How private is it, really?

Everything runs on your device. Your conversations and transcripts never leave it. The only thing MyOwnLLM does over the network is check GitHub for updates — and even that you can turn off.

Can it do what ChatGPT does?

For everyday things — yes. Drafting emails, summarising articles, answering questions, looking at photos, helping with writing or code. It's not as smart as the very largest cloud models, but it's a lot smarter than nothing, and it's yours. As local models get better, MyOwnLLM updates itself to use them.

What about my family's older laptop / the Pi in the cupboard?

That's the point. Install it everywhere — each machine gets a model that fits it. The Pi will run a small, nimble one; the gaming PC will run a big clever one. The app is the same on all of them.

For developers

OpenAI-compatible. Drop-in. Local.

MyOwnLLM also exposes an OpenAI-compatible HTTP API on 127.0.0.1:1473 (plus Ollama and Anthropic wire formats), so Cursor, Continue, Aider, Cline, Zed, Open WebUI, LibreChat, opencode, and anything else that speaks those formats works as a drop-in. Manifest-driven model selection, hardware-tier resolution, self-updating binary, fully scriptable CLI.
